Operadores em JavaScript

Operadores permitem realizar operações em variáveis e valores. Tipos de operadores:

  • Aritméticos
  • Atribuição
  • Comparação
  • Lógicos

Operadores Aritméticos

Usados para realizar operações matemáticas:

  • +: Adição
  • -: Subtração
  • *: Multiplicação
  • /: Divisão
  • %: Módulo (Resto da divisão)
  • **: Exponenciação

                // Exemplo
                let soma = 10 + 5; // 15
                let resto = 10 % 3; // 1

Operadores de Atribuição

Usados para atribuir valores às variáveis:

  • =: Atribuição simples
  • +=: Adição e atribuição
  • -=: Subtração e atribuição
  • *=: Multiplicação e atribuição
  • /=: Divisão e atribuição
  • %=: Módulo e atribuição

                // Exemplo
                let x = 10;
                 x += 5; // x = x + 5; Resultado: 15

Operadores de Comparação

Usados para comparar valores e retornam um valor booleano:

  • ==: Igual a
  • !=: Diferente de
  • ===: Estritamente igual (valor e tipo)
  • !==: Estritamente diferente
  • >: Maior que
  • <: Menor que
  • >=: Maior ou igual a
  • <=: Menor ou igual a

            // Exemplo
            let a = 5;
            let b = '5';
            console.log(a == b);  // true
            console.log(a === b); // false

Operadores Lógicos

Usados para combinar expressões lógicas:

  • &&: E lógico (AND)
  • ||: Ou lógico (OR)
  • !: Não lógico (NOT)

            let idade = 20;
            let temHabilitacao = true;

            if (idade >= 18 && temHabilitacao) {
            console.log("Pode dirigir");
            }

Exercício Prático 1

Elaborar um programa que leia um número. Calcule e informe os seus vizinhos, ou seja, o número anterior e posterior.

Exemplo: Número: 15 Vizinho: 14 e 16

Resolução Exercício Prático 1


let numero = prompt("Digite um número:");
let anterior = numero - 1;
let posterior = numero + 1;
alert("O anterior é " + anterior + " e o posterior é " + posterior);

Exercício Prático 2

Elaborar um programa para uma pizzaria, o qual leia o valor total de uma conta e quantos clientes vão pagá-la. Calcule e informe o valor a ser pago por cliente.

Exemplo: Valor da Conta R$: 90.00 Número de Clientes: 3 Valor por cliente R$: 30

Resolução Exercício Prático 2


// Valores de entrada
let valorTotal = 90.00; // Valor total da conta
let numeroClientes = 3; // Número de clientes

// Calcula o valor por cliente
let valorPorCliente = valorTotal / numeroClientes;

// Exibe o resultado formatado
console.log(`Valor por cliente R$: ${valorPorCliente.toFixed(2)}`);
