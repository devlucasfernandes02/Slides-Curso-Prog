Estruturas Condicionais

Permitem executar blocos de código com base em condições:

  • if
  • else
  • else if
  • switch

Exemplo de if/else


            let nota = prompt("Digite a nota:");

            if (nota >= 7) {
            alert("Aprovado");
            } else {
            alert("Reprovado");
            }

Exemplo de else if


            let nota = prompt("Digite a nota:");

            if (nota >= 7) {
                alert("Aprovado");
            } else if(nota >= 5 && nota < 7) {
                alert("Recuperação");
            } else {
                alert("Reprovado");
            }

Estrutura switch


            let dia = new Date().getDay();

            switch (dia) {
            case 0:
                console.log("Domingo");
                break;
            case 1:
                console.log("Segunda-feira");
                break;
            // Outros casos
            default:
                console.log("Dia inválido");
            }
