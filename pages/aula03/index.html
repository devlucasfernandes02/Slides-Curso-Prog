Estruturas Condicionais

Permitem executar blocos de código com base em condições:

  • if
  • else
  • else if
  • switch

Exemplo de if/else


            let nota = prompt("Digite a nota:");

            if (nota >= 7) {
            alert("Aprovado");
            } else {
            alert("Reprovado");
            }

Exemplo de else if


            let nota = prompt("Digite a nota:");

            if (nota >= 7) {
                alert("Aprovado");
            } else if(nota >= 5 && nota < 7) {
                alert("Recuperação");
            } else {
                alert("Reprovado");
            }

Estrutura switch


            let dia = new Date().getDay();

            switch (dia) {
            case 0:
                console.log("Domingo");
                break;
            case 1:
                console.log("Segunda-feira");
                break;
            // Outros casos
            default:
                console.log("Dia inválido");
            }

Exercício Pratico 1: Estrutura If

Crie um programa que verifica se uma pessoa pode entrar em um evento com base na idade:

  • 18 anos ou mais: Pode entrar.
  • Menos de 18 anos: Não pode entrar.

Exercício Pratico 2: Estrutura If else

Crie um programa que verifica se uma pessoa pode entrar em um evento com base na idade:

  • 18 anos ou mais: Pode entrar.
  • Se a pessoa tiver 16 ou 17 anos, exiba: "Você pode entrar, mas apenas com um responsável."
  • Menos de 18 anos: Não pode entrar.
