Loops em JavaScript

Permitem executar blocos de códigos várias vezes:

  • for
  • while
  • do while

Exemplo de loop for


            for (let i = 1; i <= 5; i++) {
            console.log("Número: " + i);
            }

Exemplo de loop while


                let contador = 1;

                while (contador <= 5) {
                console.log("Contagem: " + contador);
                contador++;
                }

Exemplo de loop do while


                let contador = 0;
                do {
                    console.log("Contador:", contador);
                    contador++;
                } while (contador < 5);

Exercício Prático 1: For

Crie um programa que conta de 1 até 5 e exibe cada número no console.

  • Use um loop for para iterar de 1 a 5.
  • A cada iteração, exiba o número atual no console.

Exercício Prático 2: While

Crie um programa que conta de 1 até 3 usando um loop while.

  • Use um loop while para iterar de 1 a 3.
  • A cada iteração, exiba o número atual no console.
