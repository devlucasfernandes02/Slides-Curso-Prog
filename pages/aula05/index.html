Funções em JavaScript

Uma função em JavaScript é um bloco de código reutilizável que executa uma tarefa específica. Ela pode receber argumentos, processar dados e retornar um valor ou apenas executar uma ação


                function nome() {
                    Bloco de Código;    
                }

Declaração de Função

Exemplo simples de declaração e invocação de função em JavaScript 


                // Declaração da função
                function saudacao() {
                    console.log("Olá, bem-vindo ao JavaScript!");
                }

                // Invocação (chamada) da função
                saudacao(); 

                Saída: "Olá, bem-vindo ao JavaScript!"

Exemplo de função sem parâmetros

saudacao() não recebe argumentos quando chamada, ela simplesmente executa o código dentro de seu bloco {}


                function saudacao() {
                    console.log("Seja Bem-vindo ao JavaScript");
                }

Exemplo de função com parâmetros

somar() recebe argumentos quando chamada, executa e retorna a soma dos valores dentro do seu bloco {}


                function somar(valor1, valor2) {
                    return valor1 + valor2; 
                }
    
                // Exemplo de chamada
                console.log(somar(5, 3));
                
                Saída: 8

Exemplo de função com parâmetros

Um exemplo de função assim é a função obterPi(), que não recebe nenhum argumento, mas retorna o valor de PI quando chamada.


                function obterPi() {
                    return Math.PI;  // Retorna o valor de pi
                }

                console.log(obterPi());
                
                Saída: 3.141592653589793
