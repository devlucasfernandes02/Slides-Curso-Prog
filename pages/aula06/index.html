Escopo de Variáveis

O escopo de variáveis em JavaScript refere-se ao contexto em que uma variável é declarada e onde ela pode ser acessada ou modificada dentro do código

E existem três tipos principais de escopo em JavaScript

  • Escopo Global
  • Escopo Local (Função)
  • Escopo de Bloco (com let e const)

Escopo Global

Variáveis declaradas fora de qualquer função ou bloco têm escopo global

                let nome = "Carlos"; // Escopo global

                function saudacao() {
                  console.log(nome); // Pode acessar a variável aqui
                }

                console.log(nome); // Pode acessar a variável aqui

               

                saudacao(); // Saída: Carlos

Escopo Local (Função)

Variáveis declaradas dentro de uma função têm escopo local àquela função Elas só podem ser acessadas dentro da função 

                function saudacao() {
                    let nome = "Carlos"; // Escopo local
                    console.log(nome); // Pode acessar a variável aqui
                }
                



                saudacao(); // Saída: Carlos
                console.log(nome); // Não pode acessar aqui

Escopo de Bloco (com let e const)

Quando usamos let ou const dentro de blocos (como dentro de loops ou estruturas condicionais) a variável tem escopo de bloco 

                if (true) {
                    let nome = "Carlos"; // Escopo de bloco
                    console.log(nome); // Pode acessar a variável aqui
                }
                
                console.log(nome); // Não pode acessar aqui

Funções Anônimas

Funções anônimas são funções sem nome. Elas são usadas frequentemente quando você precisa de uma função rápida e não precisa chamar elas por um nome

Podem ser atribuídas a variáveis, passadas como argumentos para outras funções ou até usadas diretamente na função

  • let saudacao = function() { //bloco };

Função anônima atribuída a uma variável

Aqui, a função não tem nome, mas é atribuída à variável saudacao
Você pode chamá-la usando essa variável 

                let saudacao = function() {
                    console.log("Olá, bem-vindo ao JavaScript!");
                };
                
                saudacao();  // Saída: Olá, bem-vindo ao JavaScript!

Função anônima como argumento de outra função

Funções anônimas são muito comuns quando usadas como argumentos de outras funções

                function executar(funcao) {
                    funcao(); // Chama a função passada como argumento
                }
                
                executar(function() {
                    console.log("Olá, bem-vindo ao JavaScript!");

                });  // Saída: Olá, bem-vindo ao JavaScript!

Arrow Function

Ela foi introduzida no ECMAScript 6 (ES6) e oferece uma sintaxe mais curta para escrever funções anônimas

                const soma = (a, b) => a + b;
