Exercício 1: Função sem parâmetro

Crie uma função chamada saudar que imprime
"Olá, Mundo!" no console

Exercício 1 - Resolução


            // Definição da função
            function saudar() {
                console.log("Olá, Mundo!");
            }

            // Chamada da função
            saudar();

Exercício 2: Função com parâmetros

Crie uma função chamada soma que receba dois números como parâmetros e retorne a soma deles

Exercício 2 - Resolução


            // Definição da função
            function soma(num1, num2) {
                return num1 + num2;
            }

            // Chamada da função
            console.log(soma(10,5)); 
            
            Saída: 15

Exercício 3: Função Anônima

Crie uma função anônima que seja atribuída a uma variável chamada multiplicar e que receba dois números como parâmetros e retorne o resultado da multiplicação

Exercício 3 - Resolução


// Definição da função anônima e atribuição à variável "multiplicar"
const multiplicar = function(num1, num2) {
    return num1 * num2;
};

// Chamada da função
var resultado = multiplicar(4, 5);
console.log(resultado); // Saída: 20

Exercício 4: Escopo de Variável

Crie duas variáveis, uma dentro de uma função e outra fora dela, e mostre como o escopo dessas variáveis afeta o acesso a elas

Exercício 4 - Resolução


// Variável de escopo global
var variavelGlobal = "Eu sou global!";

function testeEscopo() {
// Variável de escopo local (dentro da função)
    var variavelLocal = "Eu sou local!";

    console.log(variavelLocal); // Acesso à variável local: funciona
    console.log(variavelGlobal); // Acesso à variável global: funciona
}

// Chamada da função
testeEscopo();

console.log(variavelGlobal); // Acesso à variável global: funciona
console.log(variavelLocal); // Erro: variavelLocal não está definida fora da função
