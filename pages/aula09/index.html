Manipulando Conteúdo

Podemos alterar o conteúdo de um elemento usando:

  • element.textContent
  • element.innerHTML
  • element.value (para inputs)

// Exemplo
titulo.textContent = "Novo Título";

Manipulando Atributos

Podemos modificar atributos de elementos com:

  • element.setAttribute(atributo, valor)
  • element.getAttribute(atributo)
  • element.removeAttribute(atributo)

// Exemplo
let link = document.querySelector("a");
link.setAttribute("href", "https://www.exemplo.com");
let valor = link.getAttribute("href");
link.removeAttribute("target");
