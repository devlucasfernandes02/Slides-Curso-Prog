Manipulando Estilos

Podemos alterar o estilo de elementos via JavaScript:


// Exemplo
let titulo = document.getElementById("titulo");
titulo.style.color = "blue";
titulo.style.fontSize = "24px";
titulo.style.backgroundColor = "lightgray";
titulo.style.padding = "10px";

Alterando Múltiplos Estilos

Podemos aplicar múltiplos estilos de uma só vez usando style.cssText:


// Exemplo
let paragrafo = document.getElementById("meuParagrafo");
paragrafo.style.cssText = "color: red; font-weight: bold; background-color: yellow; padding: 5px;";

Adicionando e Removendo Classes

Manipular classes de um elemento facilita a estilização dinâmica:


// Exemplo
let elemento = document.getElementById("meuElemento");
elemento.classList.add("destaque");
elemento.classList.remove("oculto");
elemento.classList.toggle("ativo");

Verificando Classes

Podemos verificar se um elemento possui uma classe:


// Exemplo
if (elemento.classList.contains("destaque")) {
    console.log("O elemento tem a classe 'destaque'");
} else {
    console.log("O elemento NÃO tem a classe 'destaque'");
}
