Eventos em JavaScript

Eventos são ações ou ocorrências que acontecem no navegador e podem ser capturadas pelo JavaScript.

  • click
  • mouseover
  • keydown
  • load

Adicionando Event Listeners

Usamos addEventListener() para capturar eventos:


// Exemplo
let botao = document.getElementById("meuBotao");

botao.addEventListener("click", function() {
    alert("Botão clicado!");
});

Exemplo Prático - Contador

Vamos criar um contador que incrementa ao clicar em um botão.


// HTML
<button id="incrementar">Incrementar</button>
<p>Valor: <span id="contador">0</span></p>

// JavaScript
let contador = 0;
let btn = document.getElementById("incrementar");
let display = document.getElementById("contador");

btn.addEventListener("click", function() {
    contador++;
    display.textContent = contador;
});

ES6 - let e const

  • let: Declara uma variável de escopo de bloco.
  • const: Declara uma constante de escopo de bloco.

// Exemplo
let idade = 25;
const PI = 3.14;

Arrow Functions

Sintaxe mais curta para escrever funções.


// Função tradicional
function soma(a, b) {
    return a + b;
}

// Arrow function
const soma = (a, b) => a + b;

Template Literals

Permitem interpolação de expressões em strings.


let nome = "Ana";
let mensagem = `Olá, ${nome}! Seja bem-vinda.`;

Destructuring

Extrai dados de arrays ou objetos.


// Array
let [a, b] = [1, 2];

// Objeto
let { nome, idade } = { nome: "Carlos", idade: 30 };

Operador Spread

Permite expandir elementos iteráveis em locais onde zero ou mais argumentos são esperados.


let numeros = [1, 2, 3];
let outros = [...numeros, 4, 5, 6]; // [1, 2, 3, 4, 5, 6]

Exercício Prático 1

Crie uma página com um botão que, ao ser clicado, muda a cor de fundo da página para uma cor aleatória.


// JavaScript
let botao = document.getElementById("mudarCor");

botao.addEventListener("click", function() {
    let corAleatoria = '#' + Math.floor(Math.random()*16777215).toString(16);
    document.body.style.backgroundColor = corAleatoria;
});
