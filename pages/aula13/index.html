Formulários e Validação

Formulários são essenciais para coletar dados dos usuários. A validação garante que os dados inseridos sejam corretos.

  • input, textarea, select
  • Validação com JavaScript
  • Evento submit

Exemplo de Formulário

Vamos criar um formulário simples com validação:


// HTML
<form id="meuFormulario">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    <button type="submit">Enviar</button>
</form>

// JavaScript
let formulario = document.getElementById("meuFormulario");

formulario.addEventListener("submit", function(event) {
    event.preventDefault();
    let nome = document.getElementById("nome").value;
    if (nome === "") {
        alert("Por favor, insira seu nome.");
    } else {
        alert("Formulário enviado com sucesso!");
    }
});

Validação de Email

Validação básica de email usando expressões regulares:


// HTML
<form id="formEmail">
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <button type="submit">Enviar</button>
</form>

// JavaScript
let formEmail = document.getElementById("formEmail");

formEmail.addEventListener("submit", function(event) {
    event.preventDefault();
    let email = document.getElementById("email").value;
    let regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    if (regex.test(email)) {
        alert("Email válido!");
    } else {
        alert("Por favor, insira um email válido.");
    }
});

Armazenamento Local

Podemos armazenar dados do formulário no localStorage:


// JavaScript
formulario.addEventListener("submit", function(event) {
    event.preventDefault();
    let nome = document.getElementById("nome").value;
    localStorage.setItem("nomeUsuario", nome);
    alert("Nome armazenado com sucesso!");
});
