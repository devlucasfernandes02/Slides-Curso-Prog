Introdução ao JSON

JSON (JavaScript Object Notation) é um formato leve de troca de dados.

  • Fácil de ler e escrever
  • Fácil de parsear e gerar
  • Baseado em texto e independente de linguagem

Estrutura do JSON

JSON é composto por pares de chave-valor, semelhante a objetos JavaScript.


{
    "nome": "João",
    "idade": 30,
    "cidade": "São Paulo",
    "hobbies": ["ler", "correr", "viajar"]
}

Convertendo JSON para Objeto JavaScript

Use JSON.parse() para converter uma string JSON em um objeto JavaScript.


let jsonString = '{"nome": "João", "idade": 30}';
let objeto = JSON.parse(jsonString);

console.log(objeto.nome); // João
console.log(objeto.idade); // 30

Convertendo Objeto JavaScript para JSON

Use JSON.stringify() para converter um objeto JavaScript em uma string JSON.


let objeto = {
    nome: "João",
    idade: 30
};

let jsonString = JSON.stringify(objeto);

console.log(jsonString); // {"nome":"João","idade":30}

Exemplo Prático - Armazenando JSON no localStorage

Podemos armazenar objetos JavaScript como strings JSON no localStorage.


let usuario = {
    nome: "Maria",
    idade: 25
};

// Convertendo para JSON e armazenando
localStorage.setItem("usuario", JSON.stringify(usuario));

// Recuperando e convertendo de volta para objeto
let usuarioArmazenado = JSON.parse(localStorage.getItem("usuario"));

console.log(usuarioArmazenado.nome); // Maria
